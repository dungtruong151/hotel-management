# Motel Management

## Getting Started

This project is divided into two main parts: the client and the server.

### Prerequisites

- Node.js
- npm

### Running the Server

1. Navigate to the server directory:

```sh
cd server
```

2. Install the dependencies:

```sh
npm install
```

3. Start the server:

```sh
npm run dev
```

### Running the Client

1. Navigate to the client directory:

```sh
cd client
```

2. Install the dependencies:

```sh
npm install
```

3. Start the client:

```sh
npm run start
```
